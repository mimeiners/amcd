# MScEE M 2.14 Microelectronic Circuit Design (MCD)

**(c) 2024-2026 Mirco Meiners and co-authors, Hochschule Bremen - City University of Aplied Sciences (HSB), Bremen, Germany**

The course makes heavy use of circuit simulation for PCB and IC designs, using [**KiCAD**](https://www.kicad.org) for
 PCB design, [**Xschem**](https://xschem.sourceforge.io), [**KLayout**]() and
 [**SG13G2**](https://github.com/IHP-GmbH/IHP-Open-PDK) technology from [IHP
 Microelectronics](https://www.ihp-microelectronics.com) for IC design.  Both designs use
 [**ngspice**](https://ngspice.sourceforge.io) as circuit simulator.

IC Tools and PDK are integrated in the [**IIC-OSIC-TOOLS**](https://github.com/iic-jku/IIC-OSIC-TOOLS) Docker image, which
will be used during the coursework.
